# VerdiTech — Project Spec
> Project: VerdiTech | Last Updated: June 27, 2026

**Overview:** A mobile application that predicts and monitors the growth of vegetable plants using a Cellular Automata model.

**Main Features:**
1. **User Input**: Vegetable type (Tomato, Eggplant, Siling Labuyo), Planting date, Current growth stage, Season, Sunlight exposure, Water availability, Soil quality
2. **Growth Prediction System**: Uses Cellular Automata logic to predict growth stages: Seedling, Young plant, Flowering, Fruiting
3. **Plant Monitoring**: Displays health status, flags missing/insufficient factors (low sunlight, insufficient water, poor soil)
4. **Recommendation System**: Actionable tips (more sunlight, adjust watering, improve soil)
5. **Platform & Tech**: Android (iOS TBD), Flutter
6. **Expected Output**: Predicted growth stage, Plant condition, Recommended actions
