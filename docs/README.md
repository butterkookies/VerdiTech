## 🌱 About VerdiTech

**VerdiTech** is a Flutter mobile application that predicts and monitors the growth of vegetable plants (Tomato, Eggplant, Siling Labuyo) using a Cellular Automata model. It provides growth stage predictions, health monitoring, and actionable recommendations based on environmental factors.

**Client**: Senior High School student (Thesis/Capstone project)  
**Platform**: Android (Flutter) — iOS TBD  
**Core Technology**: Cellular Automata for growth simulation
