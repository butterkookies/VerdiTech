# 🎯 Mockups

UI/UX mockup files go here (Figma exports, screenshots, etc.)

---

## Expected Contents

- **Wireframes** — Low-fidelity screen layouts
- **High-fidelity mockups** — Pixel-perfect screen designs
- **Figma exports** — PNG/SVG exports from Figma design files
- **Screen flow diagrams** — Navigation and user flow maps
- **Interactive prototype screenshots** — Captures from prototype walkthroughs

---

## Naming Convention

```
[screen_name]_[version]_[fidelity].[ext]
```

Examples:
- `home_screen_v1_wireframe.png`
- `plant_detail_v2_hifi.png`
- `navigation_flow_v1.svg`
