# 📦 Assets

All design and media assets for the VerdiTech project.

---

## 📂 Subfolders

| Folder          | Purpose                                                       |
|-----------------|---------------------------------------------------------------|
| **`images/`**   | Plant images, app graphics, and general image assets          |
| **`icons/`**    | App icons (launcher icons) and in-app UI icons                |
| **`mockups/`**  | UI/UX mockup files — Figma exports, wireframes, screenshots   |
| **`references/`** | Reference images and design inspiration collected during research |

---

## 📝 Notes

- Keep assets organized in their respective subfolders
- Use descriptive filenames (e.g., `home_screen_mockup_v1.png`, not `screenshot1.png`)
- For Figma files, export as PNG/SVG and place in `mockups/`
- Original high-resolution files should be kept alongside compressed versions
